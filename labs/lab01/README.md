#  VLAN и маршрутизация между VLAN

###  Задание:

  1. Построить сеть и настроить основные параметры устройства;

  4. Настроить маршруты между VLAN на маршрутизаторе;
  5. Убедиться, что маршрутизация между VLAN работает.
